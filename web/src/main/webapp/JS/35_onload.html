DOM(Document Object Model) - onload Event Listener

DOM 객체 : document 객체를 기반으로 제공되는 HTML 태그에 대한 자바스크립트 객체

요청 웹프로그램의 실행결과를 웹문서로 전달받아 브라우저의 응답영역에 출력

브라우저는 DOM Parser를 사용하여 웹문서의 HTML 태그를 DOM Tree로 생성(꾸미기는 나중의 일) - 자바스크립트는 DOM Tree의 THML 태그를 Element 객체로 제공받아 사용 가능하다.

DOM(Document Object Model) : document 객체를 기반으로 제공되는 자바스크립트의(HTML태그의) Element 객체

중요한 이유 : 자바스크립트를 사용하여 태그에서 이벤트가 발생될 경우 이벤트 처리함수를 사용하여 태그를 Element 객체로 제공받아 조작 가능 -> 이를 이용해 동적인 페이지를 구현할 수 있다.(=DTHML : DynamicHTML)

반드시 DOM Tree가 완성된 후 동적인 페이지를 구현 가능하다. => onload 이벤트가 필요
