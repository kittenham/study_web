이벤트(event) - 이벤트 처리

이벤트 : 태그(박스모델 - Element 객체)에서 발생되는 다양한 사건

이벤트 처리(Event Handle): 태그에서 발생된 이벤트에 대한 처리를 자바스크립트 명령으로 작성하여 실행

버튼 1에 이벤트 처리를 해보자!(처리 순서)

방법1. 태그의 이벤트리스너 속성(onXXX)의 속성값으로 이벤트 처리 명령 작성

태그에서 이벤트 리스너 속성의 이벤트가 속성값으로 설정된 명령을 실행한다.

방법2. script 태그안에서 이벤트 처리함수를 생성해 호출한다.

방법3. 태그를 검색하여 Element 객체로 제공받아 Element 객체에 이벤트 리스너 속성의 이벤트 처리함수를 등록한다. -> 태그에서 이벤트 리스너 속성의 이벤트가 발생될 경우 이벤트 처리 함수가 호출된다.

방법4. 태그를 검색하여 Element 객체로 제공받아 addEventListener() 메소드를 호출하여 이벤트 처리 함수 등록 - 태그에서 addEventListener() 메소드의 이벤트가 발생될 이벤트 처리 함수를 호출한다.
